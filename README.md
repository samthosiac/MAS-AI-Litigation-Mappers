# MAS-AI-Litigation-Mappers

MAS AI Litigation Map is a web application developed for the GW Trustworthy AI Initiative Hackathon. The project aims to visualize and map litigation cases involving AI, providing users with an interactive and informative platform to explore legal trends, case details, and related data.

## Features
- Interactive map or data visualization of AI litigation cases
- Search and filter functionality for cases
- Detailed case information and summaries
- Responsive and user-friendly interface

## Getting Started
1. Clone the repository:
	```sh
	git clone https://github.com/samthosiac/MAS-AI-Litigation-Mappers.git
	cd MAS-AI-Litigation-Mappers
	```
2. Install dependencies:
	```sh
	npm install
	```
3. Start the development server:
	```sh
	npm run dev
	```
4. Open your browser and go to [http://localhost:3000](http://localhost:3000)
